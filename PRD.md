# PCAplot 产品需求文档 (PRD) v2.0

## 一、产品概述

### 1.1 产品定位

PCAplot 是一款**纯前端的 PCA（主成分分析）结果可视化与 WYSIWYG 编辑工具**。用户导入 PCA 数据文件后，自动生成高质量散点图，可对图形的每个视觉元素进行实时所见即所得编辑，最终导出为高分辨率图片或带样式的数据文件。

### 1.2 目标用户

- 生物信息学研究人员（RNA-seq、GWAS、宏基因组等 PCA 分析）
- 数据科学家与统计分析师
- 需要在论文/报告中插入高质量 PCA 图表的研究人员

### 1.3 核心价值

- **零代码**：无需编写 R/Python 作图脚本，拖入数据即出图
- **实时 WYSIWYG**：所有样式修改即时反映，无需反复渲染
- **出版级导出**：支持 72/150/300/600 DPI PNG、JPEG、SVG 矢量图，导出后可直投论文
- **可复现**：样式导出/导入（.pcastyle）完整恢复全部编辑状态

## 三、功能规格

### 3.1 数据导入

#### 3.1.1 文件格式支持

| 格式 | 分隔符 | 编码 |
|------|--------|------|
| .csv | 逗号 | UTF-8 / GBK / GB2312 / UTF-16 / ISO-8859-1 自动检测 |
| .txt | 制表符 / 逗号 / 空格 | 同上 |

#### 3.1.2 解析规则

- 自动检测表头行（若首行含非数值字段）
- 前两列为样本名和分组名，后续列为 PC1、PC2 等数值
- 缺失值自动跳过并记录
- 编码置信度低时弹出编码选择对话框（8 种编码手动切换）

#### 3.1.3 导入后自动处理

- 自动推导分组列表（按分组名列去重、排序）
- 自动分配颜色（按活跃配色方案循环）和形状（按 8 种形状循环）
- 记录文件名显示在工具栏

---

### 3.2 PCA 主图渲染

#### 3.2.1 散点图

- 每个分组一个独立的 scatter series
- 点样式：颜色、形状、大小、透明度均可全局/分组级控制
- >10,000 点时自动启用 ECharts large mode（简化数据格式以提升性能）
- 无白色描边，纯色填充

#### 3.2.2 图表元素

| 元素 | 说明 |
|------|------|
| 标题 | 默认 "PCA Plot"，可编辑文字、显示/隐藏、字号、加粗、颜色 |
| X 轴 / Y 轴 | 标题文字可编辑、字号/加粗/斜体可调 |
| 轴线 | 线宽、颜色可调 |
| 刻度 | 显示/隐藏、方向（内侧/外侧）、标签旋转（0°/45°/90°） |
| 网格线 | 显示/隐藏、实线/虚线、颜色、线宽 |
| 背景 | 白色 / 透明 |
| 点样式 | 全局大小（6-20px）、透明度（0.1-1.0） |

#### 3.2.3 形状支持（8 种）

圆形、正方形、三角(上)、三角(下)、菱形、五角星、六边形、十字

图例与侧边栏分组面板中的形状图标使用与主图 ECharts symbol 路径精确匹配的 SVG 路径，显示完全一致。

---

### 3.3 画布架构

```
pca-canvas（画板，网格点背景，可设大小）
├── chart-area-inner（PCA 主图容器，可在画板内拖拽移动）
│   └── chart-canvas-wrapper（ECharts 实例 + 8 个缩放手柄）
└── CustomLegend（图例，独立定位，可自由拖拽）
```

| 属性 | 默认值 |
|------|--------|
| 画板尺寸 | 1200 × 900 px |
| PCA 主图尺寸 | 800 × 600 px |
| 图表偏移（在画板中的位置） | X: 60, Y: 40 |
| 宽高比锁定 | 否（可开启） |
| 缩放手柄 | 8 个（四角 + 四边中点），最小 300px |

---

### 3.4 配色与样式

#### 3.4.1 配色方案（7 套）

| 方案 | 特点 |
|------|------|
| Tableau 10 | 经典 Tableau 配色，区分度高 |
| Nature 配色 | 受 Nature 期刊启发，柔和专业 |
| ColorBrewer Set1 | 经典定性配色 |
| Wong 2011 | 色盲友好，覆盖常见色盲类型 |
| Viridis 离散化 | 从连续色图均匀采样，感知均匀 |
| 高对比度 | 适合复杂图表 |
| 暖色调 | 以暖色为主调 |

每组可单独覆盖颜色。

#### 3.4.2 分组编辑（侧边栏 → 分组 标签页）

每个分组可独立设置：
- **颜色**：颜色选择器
- **形状**：8 种形状下拉选择
- **点大小**：覆盖全局大小
- **透明度**：滑块 0.1-1.0
- **组名**：双击可重命名

---

### 3.5 图例

- **位置**：独立于 PCA 主图，可自由拖拽移动
- **显示控制**：全局显示/隐藏切换
- **列数控制**：每行列数可设 1-10（1 = 单列竖向，2+ = 多列网格）
- **重命名**：双击分组名可重命名
- **形状显示**：每个图例项显示对应分组的颜色和形状图标
- **无标题**：图例不显示"图例"标题文字
- **导出可靠**：导出时通过 Canvas/SVG 手动绘制图例，100% 可靠

---

### 3.6 交互

#### 3.6.1 点选与多选

- 单击数据点：选中并显示详情（样本名、分组、PC1、PC2 坐标）
- Ctrl+点击：多选切换，信息面板列表显示所有选中点
- Esc：清除全部选中

#### 3.6.2 样本搜索

侧边栏 → 信息 → 搜索框：按样本名或分组名实时过滤，点击结果选中对应点

#### 3.6.3 离群检测

- 自动计算每组置信椭圆，检测落入其他组椭圆的离群点
- 信息面板显示警告 + 候选项列表
- >50,000 点自动跳过以保性能

#### 3.6.4 撤销/重做

- Ctrl+Z 撤销 / Ctrl+Y 重做
- 最多 50 步历史
- Toast 提示操作描述

---

### 3.7 导出

#### 3.7.1 图片导出

| 格式 | DPI 选项 | 背景 |
|------|----------|------|
| PNG | 72 / 150 / 300 / 600 | 透明 |
| JPEG | 72 / 150 / 300 / 600 | 白色 |
| SVG | 真矢量 | 透明 |

- 导出范围：**整个画板**（含 PCA 主图 + 图例），非仅 PCA 图表
- PCA 主图保留白色背景、圆角边框、阴影效果
- 图例位置精确还原用户拖拽后的实际位置

#### 3.7.2 样式导出/导入

| 格式 | 内容 |
|------|------|
| .pcastyle (JSON) | 完整样式快照（配色方案、覆盖、坐标轴、图例、画布尺寸等） |

- 导入自动恢复全部样式状态（配色方案、覆盖、尺寸、图例位置等）
- 版本兼容：旧版 .pcastyle 文件向前兼容（缺失字段自动回退默认值）

---

### 3.8 全局样式（侧边栏 → 全局样式 标签页）

| 设置项 | 说明 |
|--------|------|
| 画板尺寸 | 宽/高（400-6000px） |
| PCA 主图 | 宽/高（300-4000px）、宽高比锁定、X/Y 偏移 |
| 图表标题 | 显示/隐藏、文字、字号、加粗、颜色 |
| 图例 | 显示/隐藏、每行列数（1-10） |
| 点样式 | 全局大小（6-20px）、透明度（0.1-1.0） |
| 网格线 | 显示/隐藏、实线/虚线、颜色、宽度 |
| 背景 | 白色/透明 |
| Tooltip 字体 | 字体名、字号 |

### 3.9 适应窗口

点击工具栏"适应窗口"按钮，PCA 主图自动缩放至适合当前视口的大小。

## 四、数据流

```
导入文件 → parser（编码检测→分隔符检测→表头检测→校验）
  → useDataStore（分组推导、颜色/形状分配、离群检测触发）
  → ChartCanvas（构建 ECharts option → setOption 渲染）
  → 用户编辑 → useStyleStore（undoRedoMiddleware）
  → chartOptions 重新计算 → ECharts 增量更新
  → 导出（图片：Canvas 合成 → 下载；样式：JSON 序列化 → 下载）
```

## 五、基础设施

### 5.1 日志系统

- 分级日志（DEBUG/INFO/WARN/ERROR/FATAL）
- 内存环形缓冲（开发 1000 条 / 生产 500 条）
- 关键模块埋点（parser、store、chart、export、legend）

### 5.2 版本管理

- SemVer 版本号（MAJOR.MINOR.PATCH）
- package.json → sync-version.js → version.ts 自动同步
- 导出文件携带版本号，导入时检查兼容性

### 5.3 Toast 通知

- 工具栏右侧弹出消息
- 文件导入、导出、编辑操作均有反馈
- 错误消息可选"导出日志"
